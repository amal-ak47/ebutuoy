# Ebutouy v1.0.0 - Windows Release

## 📥 Download

**File**: `Ebutouy.exe`  
**Size**: ~300MB  
**Requirements**: Windows 10 or later  

## ✨ Features

✅ Download YouTube videos as MP4  
✅ Download YouTube videos as MP3  
✅ Download entire playlists  
✅ Select video resolution  
✅ Select audio bitrate  
✅ Automatic metadata tagging  
✅ Beautiful black & white UI  
✅ FFmpeg bundled (no installation needed)  

## 🚀 Installation & Usage

1. Download `Ebutouy.exe`
2. Run the executable (no installation needed)
3. Paste a YouTube URL (video, short, or playlist)
4. Click "Get Video"
5. Select MP3 or MP4 format
6. Choose quality/resolution
7. Click "Download"
8. Files saved to Downloads folder

## 🖥️ System Requirements

- Windows 10 or later
- 100MB free disk space
- Internet connection

## ⚠️ Troubleshooting

**App won't start**: Try running as administrator

**Downloads failing**: Check internet connection and YouTube URL validity

**Slow startup**: Normal for first run

## 📝 What's Included

- Full YouTube downloader
- FFmpeg (bundled)
- All dependencies included
- No additional installation required
